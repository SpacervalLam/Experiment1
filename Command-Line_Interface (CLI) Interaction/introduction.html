命令行交互工具

这是一个用 Python 编写的命令行工具，提供简单的交互功能。

功能介绍

  • 支持通过 -v 或 --version 参数查看工具版本。
  • 支持通过 -p 或 --prompt 参数自定义输入提示符。
  • 模拟打字机效果，逐字符输出用户输入的内容。
  • 支持处理用户通过 Ctrl+C 中断程序的情况。
  • 集成 OpenAI 大语言模型，支持通过 API 调用生成智能回复。

使用方法

python chat_cli.py [options]

可用选项：

  • -v, --version：显示工具版本号。
  • -p, --prompt：设置输入提示符（默认为 > ）。
  • -a, --apikey：设置 OpenAI API 密钥（可选，默认为环境变量 OPENAI_API_KEY）。

示例

运行以下命令查看版本：

python chat_cli.py --version

运行以下命令自定义提示符：

python chat_cli.py --prompt "输入内容: "

运行以下命令设置 API 密钥并调用大语言模型：

python chat_cli.py --apikey "your_openai_api_key"

注意事项

在运行程序时，请确保：

  • 已安装 Python 环境。
  • 已安装 OpenAI Python SDK。
  • 已设置有效的 OpenAI API 密钥（通过命令行参数或环境变量）或通过--apikey "your_openai_api_key"设置。
