命令行交互工具

这是一个用 Python 编写的命令行工具，旨在模拟打字机效果并提供简单的交互功能。

功能介绍

  • 支持通过 -v 或 --version 参数查看工具版本。
  • 支持通过 -p 或 --prompt 参数自定义输入提示符。
  • 模拟打字机效果，逐字符输出用户输入的内容。
  • 支持处理用户通过 Ctrl+C 中断程序的情况。

使用方法

python chat_cli.py [options]

可用选项：

  • -v, --version：显示工具版本号。
  • -p, --prompt：设置输入提示符（默认为 > ）。

示例

运行以下命令查看版本：

python chat_cli.py --version

运行以下命令自定义提示符：

python chat_cli.py --prompt "输入内容: "

代码实现

该工具的核心功能包括：

  1. 使用 argparse 模块解析命令行参数。
  2. 通过 simulate_typing 函数实现逐字符输出的打字机效果。
  3. 捕获 KeyboardInterrupt 异常，优雅退出程序。

注意事项

在运行程序时，请确保已安装 Python 环境，并在终端中运行命令。
